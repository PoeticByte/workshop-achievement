# workshop-achievement ｜ 新成就 New Achievement (Reburn)

> Don't Starve Together Achievement Mod —— 饥荒联机版成就养成 MOD
> 创意工坊订阅：https://steamcommunity.com/sharedfiles/filedetails/?id=2139402585

一个饥荒的成就 mod，用来增长单个存档的游戏时长、给后期一些事情做，同时让你在一个角色上体验多个角色组合的能力，并设有诸多隐藏成就。

《新成就（New Achievement / Reburn）》在原版生存玩法之上叠加了一套通用的成就、能力与等级系统：玩家通过完成成就获取点数，用于解锁各类能力或提升角色属性，从而在生存之外获得持续的成长目标与多样的强化路线。

> 本模组需服务器与全部客户端共同安装。

---

## 一、核心玩法循环

```
 生存 / 战斗 / 探索
        │
        ├─ 完成成就 ──► 成就点  ──► 解锁、升级【能力】
        │                        └► 提升【属性】(生命 / 精神 / 饥饿)
        │
        ├─ 击杀生物 ──► 杀戮值  ──► 直接完成一项未达成成就（可选）
        │
        └─ 进食 / 击杀 ► 经验 ──► 升级 ──► 获得成就点、属性点，并晋升称号
```

整套系统围绕四种资源运转：成就点、杀戮值、经验（等级）、属性点。以下分项说明。

---

## 二、四种核心资源

| 资源 | 获取方式 | 主要用途 |
| --- | --- | --- |
| **成就点** | 完成成就，每项给予 1～10 点不等 | 解锁、升级**能力**；提升**属性** |
| **杀戮值** | 击杀生物，依目标最大生命值分档计入 | 消耗固定数额，**直接完成**一项未达成成就（可在配置中开关） |
| **经验 / 等级** | 进食、击杀生物 | 升级后自动获得成就点与属性点，并晋升称号 |
| **属性点** | 每次升级获得，数量可配置（默认 1） | 提升**生命 / 精神 / 饥饿**上限 |

| --- | --- | --- |
| ≤250 | +1 | 有 33% 概率计入 |
| 250～800 | +3 | |
| 800～1250 | +5 | |
| 1250～2500 | +8 | |
| 2500～3500 | +21 | |
| 3500～5000 | +34 | |
| 5000～8000 | +55 | |
| 8000～18000 | +89 | |
| 18000～30000 | +144 | |
| 30000～55000 | +233 | |

> 蔬菜、建筑、墙体、气球、地刺等目标不计入杀戮值。
> 启用「杀戮值兑换成就」后，累计达到设定数额（默认 1000，可配置 500～5000）即可选择一项未达成成就将其直接完成，并照常发放成就点。

### 经验与升级
- **进食**：依食物档次给予经验，普通熟食约 100 点，高级料理与稀有食材更高。
- **击杀**：依目标最大生命值给予经验——99～450 点生命给 30 经验，450～900 给 30，900～2400 给 300，2400 以上给 3000，其余给 10。
- 击杀有概率掉落**经验豆**，拾取并食用后补充经验；目标越强，掉落概率越高（约 2%、9%、23%、76.5%）。
- **升级所需经验**：首级约需 1000 点，此后每升一级所需经验递增 100 点，无上限。
- 每次升级：自动获得 1 点成就点（由「人物升级」能力提供）与配置数量的属性点，并播放特效、晋升称号。

## 三、成就系统

### 规模与分类
模组内置 250 项以上成就，划分为十个分类：

| 分类 | 含义 | 示例 |
| --- | --- | --- |
| 全部 | 汇总与终极目标 | 达成全部成就 |
| 击杀 | 击杀指定生物若干次 | 击杀 100 只蜘蛛、200 只猎犬 |
| 图鉴 | 击杀首领或稀有生物 | 龙蝇、熊獾、克劳斯、远古守护者等 |
| 食物 | 食用指定料理若干份 | 食用 200 份食物、80 份培根煎蛋 |
| 制作 | 制造指定物品或建筑 | 制作 80 份火药、33 顶远古头盔 |
| 收集 | 采集或拾取指定物 | 采集 1000 次、收获 88 株仙人掌 |
| 时间 | 在特定状态下累计时长 | 零理智存活、低温或高温存活、行走或静止时长 |
| 劳作 | 砍伐、挖掘、敲击等劳动 | 砍伐 500 棵树、敲碎 60 块巨石 |
| 死亡 | 以特定方式死亡 | 被石块砸死、遭雷击、饥饿致死、毒蘑菇致死 |
| 其他 | 杂项与隐藏成就 | 克劳斯口袋、使用复活护符、收编小动物 |

### 达成方式
成就依行为类型自动判定，涵盖：击杀、进食、死亡、烹饪、制作物品或建筑、采摘、劳作、种植与部署、收编随从、向特定对象赠物、累计消耗材料，以及在特定状态下持续达成一定时长（如持有或装备指定物品、骑乘、空装备栏等）。

### 机制说明
- **难度倍率**：成就所需数量会乘以「成就完成难度」配置（0.5 / 1 / 1.5 / 2 倍）。
- **有效击杀判定**：击杀大型生物时，于其最后阶段处在一定范围内、且对其造成过伤害的玩家均判定为有效击杀，多人可共享，鼓励协同作战。
- **达成表现**：全服公告、角色头顶特效，并发放成就点；少数成就附带实物奖励（例如「美食家」奖励礼物包装蓝图）。
- **角色适配**：
  - 部分角色会屏蔽不适用的成就（例如沃姆伍德不计「毒蘑菇致死」，旺达屏蔽「被砸死、饥饿致死、遭雷击」等，沃特屏蔽多类肉食料理成就）。
  - 部分角色另设专属成就（薇诺娜、薇洛、薇克伯顿、薇格弗德、沃利、WX-78 各有一至两项专属制作成就）。

## 四、能力系统

标注「可切换」的能力，学习后可随时开启或关闭；数值强化类能力可多次学习叠加，叠加次数上限可在配置中调整。

## 五、属性系统

以**属性点**直接提升角色三项上限，每点提升 1：

| 属性 | 提升对象 | 消耗 |
| --- | --- | --- |
| 生命 | 最大生命上限 | 每 +1 消耗 1 点 |
| 精神 | 最大理智上限 | 每 +1 消耗 1 点 |
| 饥饿 | 最大饥饿上限 | 每 +1 消耗 1 点 |

- 属性点主要通过升级获得，每级获得数量可配置（1～10）。
- 可在界面中「重置属性」，按配置比例返还属性点（默认 90%，可配置 50%～100%）。
- 该系统可在配置中整体关闭。

---

## 六、重置机制

| 项目 | 说明 |
| --- | --- |
| 重置能力 | 退还已学能力，并按配置比例返还成就点（默认 90%，可配置 50%～100%） |
| 重置属性 | 退还已加属性，并按配置比例返还属性点（默认 90%） |
| 无损重置次数 | 每生存一定天数（默认 100 天，可配置 50～600）获得一次无损重置机会；能力重置与属性重置共用此次数，使用时全额返还 |

## 七、辅助系统与配置

### 开局赠礼（可关闭）
进入游戏约三秒后赠送：
- 战槌长矛与战槌头盔（非女武神本体时给予）
- 大型万圣节生命药水 ×2
- 红灯笼、南瓜灯各一

### 首领强化（可选，默认关闭）
开启后部分首领的行为将获得强化、威胁更高。例如龙蝇在命中或受击时会击落玩家的装备，并在周身召唤火环、生成荆棘陷阱（多人在场时必定触发）。该选项面向追求高难度战斗的玩家。

### 界面与操作
- 主界面分为成就、能力、等级三页，可放大或缩小。
- 按 N 键可显示或隐藏主界面（可配置）。
- 角色头顶称号随等级与进度晋升，共十一级：
  命悬一线 → 如履薄冰 → 差强人意 → 驾轻就熟 → 熟能生巧 → 得心应手 → 游刃有余 → 胸有成竹 → 挥洒自如 → 随心所欲 → 为所欲为。
  称号显示可配置开关。

## 八、上手建议

1. **开局阶段**：通过进食与击杀小型生物积累经验、提升等级，依靠「人物升级」能力稳定获取成就点与属性点。
2. **早期能力**：优先选择实用的发展型能力，如黄金矿工、砍树圣手、双倍采集、快速采集，可显著提升前期发展效率。
3. **战斗方向**：叠加提升攻击、暴击奖励与战斗嗜血，并视情况选择冰冻能力、被动反伤或坚定铠甲。
4. **属性分配**：前期侧重提升生命以保障生存，后期再补充精神与饥饿。
5. **后期发展**：成就点充裕后，可解锁角色技能移植类能力（如工程科技、灵魂跳跃、灵魂伙伴、武神军备、时间管理等），将多个角色的标志性能力集于一身。
6. **难关成就**：可使用积累的杀戮值直接完成较难达成的成就（如部分首领图鉴），以快速补充成就点。
